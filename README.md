# akka-clojure

Support akka for clojure. 

## Usage

For Leiningen:
```
[liu.mars/akka-clojure "0.1.2"]
```

## What's New

 - 0.1.4 add config-value function for convert clojure code to typesafe config
 - 0.1.2 add operators ?, ?? and !
 - 0.1.1 add state in clojure actor
 - 0.1.0 clojure actor
